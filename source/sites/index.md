# 📚 资源导航

记录一些平时会用到的网站、工具和灵感来源。  
有些是提高效率的小工具，有些是收藏很久的设计资源站，也有一些值得反复打开的网站 ✨

---

## 🖼️ 设计资源

### [World Vector Logo](https://worldvectorlogo.com/)
品牌 Logo 矢量资源下载网站，支持 SVG / PNG 格式，适合做网页设计与图标素材收集。

### [LottieFiles](https://lottiefiles.com/)
非常好用的 Lottie 动画资源站，适合做网页 JSON 动画与 Loading 效果。

### [Awwwards](https://www.awwwards.com/)
收录全球优秀网页设计作品，适合寻找 UI / 动效 / 排版灵感。

### [UIverse](https://uiverse.io/)
开源 UI 组件社区，可以直接复制按钮、输入框、动画等前端组件。

---

## 🎨 UI / 前端组件

### [Magic UI](https://magicui.design/)
基于 React + Tailwind CSS 的动画组件库，适合现代风格网页开发。

### [Sweezy Cursors](https://sweezy-cursors.com/)
提供各种创意鼠标样式与 Hover 效果，适合做个性化网站。

## 📝 PDF / 图片处理

### [iLovePDF](https://www.ilovepdf.com/zh-cn)
在线 PDF 工具箱，支持合并、拆分、压缩、格式转换等功能。

### [iLoveIMG](https://www.iloveimg.com/zh-cn)
在线图片处理工具，支持压缩、裁剪、加水印、尺寸调整。

### [YYsuni 图片工具箱](https://tool.yysuni.com/)
聚合型图片工具网站，适合快速处理日常图片需求。
